## SOCIAL MEDIA PROJECT

**PRE-REQUISITE**

- NODE.JS
- MONGODB

**INSTALLATION**

- Clone the GIT repository.

  ```bash
  git clone <project_name>
  ```

- Initialize the DB.

  ```bash
  npm run db
  ```

- Start the server.

  ```bash
  npm run devStart
  ```
